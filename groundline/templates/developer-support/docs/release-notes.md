# Release Notes

## 2026.04

The 2026.04 release introduced request id tracing for failed API calls. Error
responses now include a stable `request_id` that support can use to correlate
logs across authentication, routing, and webhook delivery systems.

## 2026.03

The 2026.03 release added dashboard filtering for webhook deliveries. Developers
can filter by event type, delivery status, endpoint URL, and retry count.

## 2026.02

The 2026.02 release improved API key management by adding last-used timestamps
and creator metadata to the developer console.
